Windows: Run the web app + optional NE Gateway (BEE)

A) Web UI only (8080)
1) Open PowerShell
2) Go to the folder (adjust if your copy path differs):
   cd "$HOME\Desktop\OneCoffeeGLB_RELEASE\web"
3) Start a local server:
   python -m http.server 8080
4) Open in browser:
   http://localhost:8080

B) BEE line (8888) — phrases /health /complete for the app
   From project root (parent of web/):
   python ne_gateway.py
   In the app, set NE Gateway base URL to http://127.0.0.1:8888 (or your cloudflared URL).

PWA install:
- Chrome/Edge: Install icon in address bar.
- iPhone: Share → Add to Home Screen (PWA behavior depends on Safari).

Note: Service Worker requires http://localhost or https.
