NE Gateway 回線（ne_gateway.py）— アプリとは別プロセス

起動（プロジェクトルート）
  python ne_gateway.py
  python ne_gateway.py --port 8888

疎通（ローカル）
  curl -s http://127.0.0.1:8888/
  curl -s http://127.0.0.1:8888/health
  curl -s http://127.0.0.1:8888/phrases
  curl -s "http://127.0.0.1:8888/api/phrases"
  curl -s http://127.0.0.1:8888/api/templates/hotel
  curl -s http://127.0.0.1:8888/api/dict/bootstrap
  curl -s -X POST http://127.0.0.1:8888/api/translate -H "Content-Type: application/json" -d "{\"text\":\"hello\",\"from\":\"en\",\"to\":\"es\"}"
  curl -s "http://127.0.0.1:8888/complete?case_id=demo-001"

外部（Cloudflare Quick Tunnel）
  cloudflared tunnel --url http://127.0.0.1:8888
  → 表示された https://....trycloudflare.com をメモし、上記パスを置き換えて確認

データ
  data/generated_phrases.json … GET /phrases /api/phrases
  templates/*.json … GET /api/templates/<name>
  evidence.db … /complete（SQLite）
